文件管理应用使用说明

功能特性：
1. 文件上传（支持批量上传、文件夹上传）
2. 文件管理（排序、筛选、搜索）
3. 文件查看（PDF/HTML/图片等格式）
4. 文件操作（删除、批量删除、按后缀删除）
5. 收藏功能（收藏文件、查看收藏文件）
6. 分页功能（支持大量文件浏览）

如何运行：
方法1：使用Python内置服务器
- 打开命令提示符
- 进入当前目录
- 运行：python -m http.server 8081
- 在浏览器中访问：http://localhost:8081

方法2：直接打开HTML文件
- 双击index.html文件
- 在浏览器中打开

主要功能使用：
- 上传文件：点击"上传文件"按钮选择文件或文件夹
- 搜索文件：在搜索框中输入关键词
- 筛选文件：使用类型筛选下拉菜单
- 排序文件：使用排序下拉菜单
- 收藏文件：点击文件项右侧的★按钮
- 查看收藏：点击头部的"收藏夹"按钮
- 批量操作：使用选择框和批量操作按钮

注意事项：
- 支持批量上传最多700个文件
- 支持文件夹上传（选择文件夹时会上传所有子文件）
- 文件数据保存在浏览器本地存储中
- 建议使用现代浏览器（Chrome、Firefox、Edge等）
